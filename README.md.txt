# Spider - CTF Crypto Challenge Analyzer & Interaction Framework

Spider is an intelligent tool designed for CTF players to analyze cryptographic challenges, process multiple input files, extract meaningful signals, detect crypto structures, suggest solving strategies, and interact with live challenges (netcat).

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python spider/cli/spider.py inspect <files>
python spider/cli/spider.py solve <files>
python spider/cli/spider.py nc <host> <port>
```
